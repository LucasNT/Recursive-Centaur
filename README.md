# Recursive-Centaur
Recursive Centaur in python using a recursive function.

Example:

```bash

 o            0
/|\_____
|       #
! !  ! !

 o            1
/|\_____
|       #____
! !  |       #
     ! !  ! !

 o             2
/|\_____
|       #____
! !  |       #____
     ! !  |       #
          ! !  ! !     

```
